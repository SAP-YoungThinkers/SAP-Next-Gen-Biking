# Sending trajetories to HANA
## JSON Format
Collected data should be formatted in JSON like in the following scheme:
```json
{
    "trackPoints":
        [
            [47.667179, 9.168659, 1481625334952],
            [47.667180, 9.168669, 1481625334953]
        ]
}
```
